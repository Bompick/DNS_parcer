# DNS Pricing Scraper

Этот проект предназначен для сбора цен на телевизоры LG с сайта DNS Shop.

## Требования

- Python 3.8+
- Google Chrome (последняя версия)
- Установленные библиотеки из `requirements.txt`

## Установка

1.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

## Запуск

1.  Запустите скрипт:
    ```bash
    python DNS_pricing/scrape_dns.py
    ```

Скрипт автоматически:
- Откроет скрытый экземпляр Chrome.
- Подберет подходящую версию драйвера.
- Перейдет на страницу каталога LG TV.
- Соберет данные (название, цена, наличие).
- Сохранит результат в файл Excel с текущей датой и временем.
- Цены отсутствующих товаров будут помечены оранжевым цветом.

## Устранение неполадок

- **Ошибка версии Chrome**: Если скрипт падает с ошибкой версии драйвера, убедитесь, что у вас установлен последний Chrome, и попробуйте запустить скрипт снова — он попытается скачать нужный драйвер.
- **Блокировки**: Сайт DNS имеет сильную защиту от ботов. Используется библиотека `undetected-chromedriver`. Если она перестает работать, возможно, сайт обновил защиту.
